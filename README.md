## About

This repository stores web technology lab programs and basic browser-based exercises. The files are mostly small, focused examples intended to demonstrate one concept at a time.

## Repository Map

- `Basic Programs/`: navigation bar, XML, forms, validation, calculator, login/signup, background image behavior, and simple arithmetic pages.

## Use

Open the `.txt` examples as source references, copy the relevant HTML/CSS/JavaScript into a browser-ready file if needed, and run them locally in a browser.


## Getting Started

Clone the repository:

```bash
git clone https://github.com/CoderFatherBB/Web_Technology.git
cd Web_Technology
```

Open the folder or notebook you want to explore. Most directories are self-contained, so start from the local `README.md`, source file, notebook, or project folder for that topic.
